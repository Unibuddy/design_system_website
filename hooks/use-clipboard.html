useClipboard

If you need an easy way to copy some text in to the clipboard, use the useClipboard hook. It exposes two variables canUse and copyToClipboard. This functionality is not available on legacy platforms so make sure it is available before allowing the user to take action.

Beware, copyToClipboard is an async function and can fail for some arbitrary reason. It is a good practice to always wrap your async calls in to an awaitable try/catch block.

Be nice to your users and show a little Toast as a sign of success.
