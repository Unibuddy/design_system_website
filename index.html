Unibuddy Patron

Patron is a Design System for Unibuddy apps. Its purpose is to help you develop products and features fast, while maintaining the look and feel of the Unibuddy brand across all of our products, and help enforce the quality standards of accessibility and usability for all users.

Get Started

To start using the package first install it as a dependency.

Then add peer dependencies

Then wrap your entire app in the DS provider.

Next, we have to import the patron css. Usually this would be done in the App.js but due to some temporary issues with our tooling, this must be done in a .css file. Preferably, App.css. Important to note is that this cannot be a .pcss or .module.css file.

In App.jsx we add the App.css file, if we don't have one already.

Next up, we import patron css inside that css file.

Finally, import the DS components in your application code
