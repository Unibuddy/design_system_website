ToggleSection

Less opinionated with styling, and more flexible api than Collapsible, this component allows you to build a more custom Collapsible components but requires more work upfront. If the Accordion and Collapsible don't work for you, use this component instead.

How To Use

Wrap your children with the ToggleSection and give it a label. The label can be any ReactNode. It is an controllable component, so the state of the expanded property can be controlled both internally, or passed in from above. Kind of like a native input behaves.

To control the expanded property from above, pass it the isOpen and onToggle props.

Default State

If you only want to control the initial state of the expanded property but then let the component control its internal state, you can use the defaultState prop.
