Pagination

The purpose of this component is to provide an accessible interface for pagination controls. It provides an additional helper hook if you wish for it to control your data for you internally.

Props

limit (required)

The number of data items you want to show in one section.

offset (required)

The index offset. You can pass your own value here or one from the helper hook. The controls from the component will change the offset based on the limit provided.

itemCount (required)

The total number of data items.

renderButton (required)

This returns a callback that you may use to access some of the props Pagination returns. The props returned are:

  1. offset - you may use this to control the data based on the offset
  2. isActive - you may use this to control the color based on active states
  3. disabled - if the button is disabled or not (applies to the "Previous" and "Next" buttons)

You can pass a component that you want rendered as controls. Usually Button or ButtonLink

Example

usePagination hook

Patron also provides a helper hook if you don't want to worry about managing data yourselves. It takes the list you want pagination and the limit as params and returns paginatedData , offset and setOffset which you can simply pass to the component.
