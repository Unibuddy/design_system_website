ConfirmTextPrompt

ConfirmTextPrompt is a high level component that is used to quickly add confirmation actions in to the ui for very destructive actions. It encourages some best practices, and if it does not meet your requirements, you should probably use the Dialog component.

Example usage

<ConfirmTextPrompt
    isOpen={this.state.isOpen}
    onConfirm={this.onConfirm}
    onDismiss={this.onDismiss}
    title="Are you sure you want to delete this?"
    type="danger"
    confirmText="Yes, delete forever"
    requireInput="name of the thing"
>
    <p>Are you sure you want to permanently delete this thing. This action cannot be reversed!</p>
</ConfirmTextPrompt>
