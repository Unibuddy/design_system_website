TextArea

TextArea is a lightweight wrapper around the native html textarea. In most cases it should be used together with FormGroup inside the Form component.

Controlled use

As the native element, TextArea accepts onChange and value props.

Use in a Form

To use with a Form, wrap TextArea with a FormGroup component and give it a name prop.
